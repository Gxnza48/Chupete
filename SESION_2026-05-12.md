## Qué se construyó esta sesión (Backlog v0.2 completo)

### Features implementadas
1. **Venta Rápida** — botón en `ItemPreviewModal`, API `/api/quick-sell`, precios fijos por rareza desde `lib/quickSell.ts`
2. **Leaderboard** — `/leaderboard` server component, 3 tabs (Clickers/Coleccionistas/Traders), SQL functions en migration 005, avatares + marcos + charms + badges de nivel
3. **Items Limitados** — columnas `available_from`/`available_until`/`is_limited` en `items` (migration 006), filtrado en open-case
4. **Upgrade** — `/upgrade` con rueda circular CS2-style (arco SVG rotante), framer-motion, odds por pasos de rareza (`lib/upgrade.ts`), API `/api/upgrade`
5. **Crafteo** — `/crafteo` multi-select 2-5 items, tabla de outcomes por valor (`lib/craft.ts`), API `/api/craft`
6. **Condición + Desgaste** — columna `durability`/`max_durability` en inventory (migration 007), barra de color en ClickerArea (verde→lima→amarillo→naranja→rojo), edge function reduce durabilidad por click y rompe el item en 0
7. **Subastas** — tablas `auctions`/`auction_bids` (migration 008), página `/subastas` con create/pujar/reclamar, fee 5%
8. **Autoclicker** — columnas en profiles (migration 009), `AutoclickerPanel` en pantalla principal, `/api/buy-autoclicker` + `/api/claim-autoclicker` (acumula XP solo, SIN créditos)

### Dopamina / efectos
- **Caja**: fondo rarity-color al abrir, partículas, anillos expansivos, rayos SVG (extraterrestre/en el ort), screen shake (ultra_raro+), sonidos progresivos que desaceleran (`startSpinTick`)
- **Clicker**: efectos por rareza — GoldSparkles (legendario), CyanZap (extraterrestre), CosmicBolt zigzag (en el ort); barra de durabilidad debajo del chupete; sonido de click via Web Audio API
- **Upgrade/Craft**: toast de drop al ganar, sonido progresivo al girar la ruleta, sonido de win/lose
- **Sonidos**: `lib/sounds.ts` — Web Audio API puro, sin archivos externos

### Sistema de sonidos (`lib/sounds.ts`)
- `playClick()` — tick corto en cada click
- `startSpinTick(duration)` — ticking progresivo que desacelera (55ms→600ms), retorna stop fn
- `playCaseResult(rarity)` — acorde polifónico, más notas = más rara
- `playUpgradeWin/Lose()` — fanfarria o bajón
- `playCraftSuccess/Fail()`

## Edge function roll-drop — lógica actual

- 0.1% drop chance por click
- Créditos: SOLO de drops raros, level ups, milestones (NO por click normal)
- Durabilidad: reduce por `clickCount` al item equipado; item se rompe (delete) cuando llega a 0
- Item equipado: `profiles.equipped_chupete_id` → `inventory.id`
